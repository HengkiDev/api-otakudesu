Anime API

Access anime information from Otakudesu with our easy-to-use API endpoints.

Get Ongoing Anime

GET /api/anime/otakudesu/ongoing

Get Complete Anime

GET /api/anime/otakudesu/complete

Search Anime

GET /api/anime/otakudesu/search/:query

Example:

/api/anime/otakudesu/search/naruto

Get Anime Details

GET /api/anime/otakudesu/detail/:endpoint

Example:

/api/anime/otakudesu/detail/one-piece
